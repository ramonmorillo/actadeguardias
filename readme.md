# Farmacia Guardias

Aplicación interna para registrar incidencias de guardias de fin de semana del Servicio de Farmacia.

## Objetivos

- Registrar partes de guardia
- Añadir incidencias múltiples
- Buscar incidencias
- Generar informes entre fechas
- Histórico
- Estadísticas
- Multiusuario

## Stack previsto

- Google Apps Script
- Google Sheets
- Google Drive
- GitHub para versionado
